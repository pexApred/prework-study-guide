Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • Metadata is data about our webpage that the browser uses to effectively serve our webpage to users and is contained in the head.
  • The body element represents the visible content shown to the user.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage.
  • An element refers to the entire element from opening tag to closing tag, i.e. what appears inside the angle brackets.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain the property and value for that property.
  • Two or more elements that will share the same styling can be grouped together with a comma.
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector. Then the class attribute can be assigned to the HTML, enabling us to choose freely which elements will be assigned the CSS rule.
